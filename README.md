# Software_Engineering_Labs
This repository contains the software engineering lifecycle artifacts for a Home Temperature Adjustment System, developed as part of the Software Engineering course.

Project Overview
The project focuses on building an intelligent climate control system that optimizes energy consumption based on room occupancy. By utilizing sensors and user-defined preferences, the system ensures comfort while minimizing waste.

Key Objectives
Occupancy-Based Control: Automating temperature based on real-time presence.

Energy Efficiency: Reducing costs through smart scheduling and sensor data.

Software Design: Applying rigorous engineering principles, from requirements to final design.

Repository Structure
The project is organized into two main branches:

main: Contains the final versions of documentation and project overviews.

Labs: Includes weekly progress, laboratory tasks, and iterative design steps.

Branch Structure:
├── Project/              # Project Branch 
├── Srs_Document/         # Srs Document For Project
├── Lab/                  # Weekly laboratory Things
