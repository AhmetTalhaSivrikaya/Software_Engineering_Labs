# Software_Engineering_Labs
This branch contains the laboratory applications and development phases for the Software Engineering course.

Purpose
The primary goal of these labs is to implement software engineering principles—such as requirement analysis, UML modeling, and software design—by developing a Smart Home Temperature Control System.

Repository Structure
The organization of the Lab files is as follows:

Software_Engineering_Labs (Labs Branch)
├── Lab1/                  # Initial assignments and introduction
├── Lab2/                  # Requirement analysis and specifications
├── Lab3/                  # UML and design files
│   ├── UseCase_Diagrams/  # System use case diagrams
│   └── Class_Diagrams/    # System class architecture
├── Documentation/         # Project reports and Design Documents (SDD)
└── README.md              # Project overview

Content Summary
Analysis: Functional and non-functional requirements defining the system behavior.

Design: Architectural visualization using various UML diagrams.

Documentation: Detailed technical reports covering the software design process.
